# homebrew-tap

Personal Homebrew tap.

## Install

```bash
brew install YOUR_GITHUB_USER/tap/claude-agents
```

Or tap first:

```bash
brew tap YOUR_GITHUB_USER/tap
brew install claude-agents
```

## Formulae

- **claude-agents** — local multi-agent dev setup: AI code review, test guardian,
  and cross-repo PR-lesson mining. Source: https://github.com/YOUR_GITHUB_USER/claude-dev-agents
